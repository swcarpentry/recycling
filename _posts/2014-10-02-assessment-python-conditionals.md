**Question 1:**  
What would this Python code snippet print?

<pre>foo=10
bar=10
baz=20
if (foo*2!=baz):
  print "a"
elif (baz or foo&lt;baz):
  print "b"
elif (foo&lt;=baz and foo+bar==baz):
  print "c"
else:
  print "d"</pre>

Answers:

*   a
*   b
*   c
*   d
