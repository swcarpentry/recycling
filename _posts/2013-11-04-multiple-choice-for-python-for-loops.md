**Novice question**. Consider the following code:

<pre>instructors = ['Anna', 'Critch', 'Julia']

n = 0
for inst in instructors:
    print(inst)
    n += 1
print(n)</pre>

This prints out:

a)

Anna, Critch, Julia  
3

Critch  
2  
Julia

Julia  
3

Anna  
Critch  
Julia  
2
